# Crypto Momentum App

A real-time cryptocurrency scanner for detecting momentum patterns and trading opportunities.

## Features

- Real-time price monitoring
- Momentum detection
- Volume analysis
- Technical indicators
- Alert system

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from scanner import CryptoScanner

scanner = CryptoScanner()
scanner.start()
```

## Configuration

Edit `config.py` to customize:
- API endpoints
- Scan intervals
- Alert thresholds
- Cryptocurrency pairs
